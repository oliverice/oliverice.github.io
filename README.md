# Clean Personal Website

A simple one-page personal website built with HTML and CSS.

## How to use

1. Open `index.html` in your browser.
2. Edit the text in `index.html`.
3. Edit the design in `style.css`.

## Files

- `index.html` — website content
- `style.css` — website design
- The website is available at https://oliverice.github.io
